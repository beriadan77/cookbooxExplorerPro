# Cookbook Explorer Pro

Cookbook Explorer Pro es una aplicación hecha con **Vue**, **Vue Router** y **Pinia**. Utiliza la API de TheMealDB para explorar recetas, guardar favoritas y planificar comidas semanales.

## Vistas principales

La aplicación tiene tres vistas principales:

- Home.
- Recipes.
- Planner.

## Home

La vista Home está dividida en 3 secciones:

A. Receta random del día.  
B. Sección favoritos.  
C. Sección del planner.  

## Funcionalidad

- Se da “Me gusta” a una receta desde la página de receta o desde la grilla de recetas.
- La receta aparece en favoritos.
- Las recetas favoritas pueden usarse en el Planner.
- En los slots del planner se elige qué receta favorita incluir.

## Tecnologías

- Vue.
- Vue Router.
- Pinia.
- TheMealDB API.

## Mejoras previstas

- Agregar botón de deshacer al eliminar una receta de favoritos.
- Ordenar los favoritos en secciones de desayuno, almuerzo, etc., según hayan sido asignadas a los slots del planner.
- Agregar exportación a iCal para añadir el calendario a una app de calendario móvil.
